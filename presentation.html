Remixing Questions

questions.json

  • A list of question objects
  • Each question object has three properties:
    • question - the question you are asking
    • answer - URL to the picture that is the correct answer
    • theme - Flickr search term for wrong answers

Example


[
  {
    "question": "Which one is a kitten?",
    "answer": "http://static.comicvine.com/uploads/original/13/132956/3181555-cute-kitten-kittens-16122946-1280-800.jpg",
    "theme": "puppies"
  },
  {
    "question": "Which one is a puppy?",
    "answer": "http://puppywallpapers10.net/wp-content/uploads/images/c9/puppies-1.jpg",
